Gameplay

Three difficulty levels — Easy (4×4), Medium (4×5), Hard (5×6)
40 unique emoji symbols, randomly shuffled each game
Cards flip with a smooth 3D rotation animation
Matched pairs glow green and stay revealed

Tracking

Live move counter
Stopwatch timer (starts on first game)
Pairs found counter (e.g. 6/10)
Animated progress bar fills as you find matches

Win Screen

Celebration modal shows your final moves + time
One-click "Play Again" restarts with a fresh shuffle
